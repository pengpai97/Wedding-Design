# Wedding Design - 婚礼座次表

移动端友好的婚礼宴席座次表网页应用。

## 功能特点

- 📱 完美适配手机屏幕
- 🔍 宾客姓名搜索功能
- 💒 点击桌号查看详细宾客名单
- 🎨 优雅的婚礼主题设计

## 访问地址

启用GitHub Pages后访问：
```
https://pengpai97.github.io/Wedding-Design/
```

## 本地预览

```bash
# 启动本地服务器
python -m http.server 8080

# 访问
http://localhost:8080/
```

## 文件说明

- `index.html` - 婚礼座次表主页面
- `婚礼座次表.html` - 座次表源文件
- `update_seating.py` - 座次表更新脚本
- `.nojekyll` - GitHub Pages配置文件
